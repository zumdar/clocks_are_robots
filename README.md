**Arduino Pinouts**

A0 - Audio in <br>
A1 - Enveloped audio in<br>
A2 - Octave knob <br>
<br><br>
[D4 - D7] are for stepper motor<br>
D4 - AIn1<br>
D5 - Ain2<br>
D6 - Bin1<br>
D7 - Bin2<br>
D8 - Start/Stop Button<br>
D9 - Tone out <br>
D10 - Servo out<br>
D12 - Neopixel LED out<br>
