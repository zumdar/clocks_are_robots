**Arduino Pinouts**

A0 - Audio in
A1 - Enveloped audio in
A2 - Octave knob 

[D4 - D7] are for stepper motor
D4 - AIn1
D5 - Ain2
D6 - Bin1
D7 - Bin2
D8 - Start/Stop Button
D9 - Tone out 
D10 - Servo out
D12 - Neopixel LED out
